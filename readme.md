#### Description
The C++ ExprTk Extras are a series of additional examples demonstrating
in more detail various aspects of ExprTk and its usage.


#### Download
http://www.partow.net/programming/exprtk/index.html


#### Compatibility
The ExprTk Extras implementation is fully compatible with the following C++ compilers:
+ GNU Compiler Collection (4.1+)
+ Intel® C++ Compiler (9.x+)
+ Clang/LLVM (1.1+)
+ PGI C++ (10.x+)
+ Microsoft Visual Studio C++ Compiler (8.1+)
+ IBM XL C/C++ (10.x+)
